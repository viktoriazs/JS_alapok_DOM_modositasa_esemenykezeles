JavaScript gyakorló feladatok

Az alábbi feladatokat mind külön metódus hívásával valósítsd meg!

Elemek elérése 1

Írj JavaScript kódot, amely kiírja a konzolra az ehhez a bekezdéshez tartozó címben lévő szöveget! (azaz az első section h2-tagjében lévő szövget).

Segítség a megoldáshoz:

  • Az első section elem elérése: const ELEM=document.querySelectorAll("section h2")[0]
    ld: querySelectorAll
  • Elem tartalmának kiolvasása: ELEM.innerHTML ld.: innerHTML
  • Konzolra írás: console.log()

Elemek elérése 2

Írj JavaScript kódot, amely az "#ide" azonosítóval ellátott elembe beletesz egy p taget, és a p tag tartalma "Jó reggelt!"

Segítség a megoldáshoz:

  • Az első section elem elérése: const ELEM=document.querySelectorAll("#ide")[0], vagy
    const ELEM=document.getElementByID("ide")     ld.: getElementByID
  • Elem tartalmába írás: ELEM.innerHTML

Elemek elérése 3

Írj JavaScript kódot, amely az ".ide" azonosítóval ellátott elembe beletesz egy p taget, és a p tag tartalma "Jó reggelt!"

Segítség a megoldáshoz:

  • Az első section elemelérése: const ELEM=document.querySelectorAll(".ide")[0],
    vagy const ELEM=document.getElementsByClassName("ide")     ld.: getElementsByClassName
  • Elem tartalmába írás: ELEM.innerHTML

Elemek elérése 4

Írj JavaScript kódot, amely a "lista" azonosítóval ellátott elembe beletesz felsorolást a felsorolás tartalma 10 és 30 közötti véletlen számok legyenek, összesen 5 db!

Segítség a megoldáshoz:

  • Véletlen szám generálása: Math.random() - [0,1) között generál számokat.
    Math.floor() - levágja a tört részt.     ld.: getElementByID
  • Érdemes egy szöveges változóban összeállítani a html kódot, majd azt írni bele a megfelelő elembe.
    Tartalom összefűzése, ha változókat is használunk: (alt+gr+7) `szöveg ${valtozonev} szöveg`

Segítség a megoldáshoz:

Eseménykezelés 1

Írj JavaScript kódot, amely az előző sectionban lévő "lista" azonosítóval ellátott elemhez hozzáad egy eseménykezelőt! Az elemre való kattintáskor az ebben a sectionban lévő "kattintasutan" div-be kerüljön bele a lista div tartalma.

Segítség a megoldáshoz:

  • Eseménykezelő hozzáadása ELEM.addEventListener("esemeny", fvenynev).
  • A lehetséges eseményekről itt tájékozódj:HTML DOM events

Eseménykezelés 2

Írj JavaScript kódot, amely az ebben a sectionban lévő "feladat" azonosítóval ellátott elembe elhelyez egy gombot. A gombra kattintva pedig ugyanabba a divbe a gomb mellé még hozzáad egy újab divet, amiben egy kép van.

Segítség a megoldáshoz:

  1. "Fogd meg" a feladat elemet. (ld.:1. feladat )
  2. Az elembe írd bele a gombot: "<button>OK</button>" (ld.:2. feladat )
  3. "Fogd meg" a gomb elemet: (".feladat button") (ld.:1. feladat )
  4. Adj a gomb elemhez eseménykezelőt (ld.: Eseménykezelés 1 ).
  5. Az eseménykezelő függvényében írd bele a a feladat elembe a divbe ágyazott képet tartalmazó html kódot. (ld.:2. feladat )

Eseménykezelés 3

Írj JavaScript kódot, amely az előző a sectionban lévő "feladat" azonosítóval ellátott elemben lévő képre rátesz egy eseménykezelőt. Ha a kép fölé visszük az egeret, akkor a kép mérete változzon meg! Ha levisszük a képről az egeret, akkor álljon vissza az eredeti méret.

Segítség a megoldáshoz:

  • Egymás után több eseménykezelőt is adhatsz egy elemhez.
  • A lehetséges eseményekről itt tájékozódj:HTML DOM events
  • Az elemeket formázhatod előre elkészített css osztályokkal, ld.: Elemek formázása 1 feladat

Eseménykezelés 4

Írj JavaScript kódot, amely az ebben a sectionban lévő "tarolo" azonosítóval ellátott elemben lévő divekre rátesz egy-egy eseménykezelőt. kattintásra az elem tartalmát írja bele az "eredmeny" divbe!

Segítség a megoldáshoz:

  • Ha több elemre akarod rátenni ugyanazt az eseménykezelőt, akkor egy ciklussal be kell járnod az elemeket!
  • Az event.target utasítás adja vissza annak a eseményt kiváltó elemet.
